Проанализируй, насколько кандидат подходит для указанной вакансии. На основе полученных данных сформируй карточку кандидата. Карточка должна включать следующие пункы:

В качестве заголовка напиши "Резюме ФИО". Если ФИО не указано напиши "нет данных". Спец символы в этой строке не используй, только буквы.

1. ФИО кандидата
2. Краткая сводка по резюме
3. Краткое резюме результатов личностного теста (PAEI), с оценкой подходит ли кандидат
4. Краткий вывод по выполнению тестового задания кандидата.
5. Сравнение с требованиями вакансии.
6. Итоговая оценка: Подходит / Условно / Не подходит с кратким объяснением
7. Оценка по шкале от 1 до 10, в формате 1/10

Если каких-то данных нет - укажи это в оценке.
Обязательно поясняй причины оценки для каждого пункта вакансии.
Любое несоответствие ключевых навыков, технологий, должностных обязанностей или уровня владения языками = балл снижается.
